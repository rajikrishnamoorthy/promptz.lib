# Creating a New Presentation

Follow this workflow when creating a new PowerPoint presentation from scratch.

## Step-by-Step Process

### 1. Initialize the Presentation
- Use `create_presentation` to start a new blank presentation
- Consider using a template if consistent branding is needed

### 2. Structure Your Content
Before adding slides, plan the structure:
- Title slide (required)
- Agenda/Overview slide (recommended)
- Content slides
- Summary/Conclusion slide
- Q&A or Contact slide (optional)

### 3. Add the Title Slide
- Use `add_slide` with a title layout
- Add main title with `add_title`
- Add subtitle or presenter name

### 4. Build Content Slides
For each content section:
- Use appropriate slide layout
- Add clear, concise headings
- Use bullet points for lists (max 5-6 points)
- Add supporting visuals

### 5. Add Visual Elements
Enhance engagement with:
- Charts for data (`add_chart`)
- Images for concepts (`add_image`)
- Tables for comparisons (`add_table`)
- Shapes for diagrams (`add_shape`)

### 6. Apply Formatting
- Ensure consistent fonts and sizes
- Use brand colors if applicable
- Check text alignment and spacing

### 7. Save and Review
- Always use `save_presentation` with a clear filename
- Include .pptx extension
- Verify the file was created successfully

## Tips for Effective Presentations

- **One idea per slide** - Keep content focused
- **6x6 rule** - Max 6 bullet points, 6 words each
- **High contrast** - Ensure text is readable
- **Consistent style** - Use same fonts and colors throughout
- **Quality images** - Use high-resolution graphics
